Base for agent-based log messages.

This class adds the object variable 'log' to each instance of a subclass that inherits this class. This log variable is a logging object that is to be used for creating log messages.

Initialises the logger.

The logger is initialised to send log messages under the logging domain [AgentId].[log_name]. The name of the agent is retrieved by accessing agent.id variable.

If the logger is initialised for the agent itself, logName has to be set to an empty string.

Parameters:
  • agent (POSH.AgentBase) - A POSH agent.
  • log_name (string) - Name of the logging domain, "" if called for the agent.
  • default_level (int) - The default logging level.
