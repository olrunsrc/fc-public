fire(self)

source code 

Fires the drive collection.

This method first checks if the goal (if not None) is met. If that is the case, then FireResult(False, self) is returned. Otherwise it goes through the list of priority elements until the first one was fired successfully (returning something else than None). In that case, FireResult(True, None) is returned. If none of the priority elements were successful, FireResult(False, None) is returned, indicating a failing of the drive collection.

To summarise:

  • FireResult(True, None): drive element fired
  • FireResult(False, self): goal reached
  • FireResult(False, None): drive failed
