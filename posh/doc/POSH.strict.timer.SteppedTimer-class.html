Class SteppedTimer

source code

TimerBase --+
            |
           SteppedTimer

A stepped agent timer.

This timer is a stepped timer, which is to be used if the agent is stepped, i.e. controlled from and outside controller. The timer starts at time 0 and increases the time every time that loopEnd is called. It does not provide loop timing, as that wouldn't make any sense if the agent is controlled from the outside.
