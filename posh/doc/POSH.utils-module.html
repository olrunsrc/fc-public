is_library(library)

source code 

Returns if the given library is a valid behaviour library.

This is the case if the following requirements are met:

  • there is a directory with the given name in LIBRARYPATH
  • the directory can be loaded as a module, which requires it to at least contain a __init__.py file
Parameters:
  • library (string) - Name of the library to check
Returns: bool
If the library is a valid library

get_behaviours(library, log=None)

source code 

Returns a sequence of classes, containing all behaviour classes that are available in a particular library.

The method searches the behvaiour subclasses by attempting to import all file in the library ending in .py, except for the WORLDSCRIPT, and search through all classes they contain to see if they are derived from any behaviour class.

If a log object is given, then logging output at the debug level is produced.

Parameters:
  • library (string) - Name of the library to find the classes for
  • log (as from the Python logging module) - A log object
Returns:
The list of classes

run_world_script(script_file, library, world_args=None, agents_init=None)

source code 

Runs the given file to initialise the world and returns the world object and if the world initialisation script creates and runs the agents itself.

This method creates an instance of the class World and makes it accessible to the world initialisation script under the instance name 'world'. Using this instance, the world intialisation script can cimmunicate with this function.

Parameters:
  • script_file (string) - The filename of the script to run
  • library (string) - name of the behaviour library to use by agents
  • world_args (string) - arguments given to the world initialisation script
  • agents_init (as returned by POSH.agentinitparser.parse_agent_init_file) - agent initialisation information structure
Returns: (any, bool)
tuple (world object, if script created and ran agents)
Raises:
  • IOError - If it cannot file the script
  • Exception - If the script causes an exception
