Class LAPParser

source code

A recursive descent parser for .lap files.

The parser takes a single input string that represents the plan file and creates a plan builder object that it then returns.

If an error while parsing (or tokenising) is encountered, a ParseError is raised.

match(self, allowed_tokens)

source code 

Checks if the current token matches the allowed tokens.

If there is no current token, then this method raises an Exception that indicates that we've reached the end of the input (unexpectedly).

Otherwise it returns if the current token type matches any of the given token types.

Parameters:
  • allowed_tokens (list of string) - A list of allowed tokens.
Returns: boolean
If the current token type matches any of the allowed tokens.
Raises:
  • ParseError - If the is no current token.
