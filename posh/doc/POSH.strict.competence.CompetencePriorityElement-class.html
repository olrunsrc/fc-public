fire(self)

source code 

Fires the competence priority element.

This method goes through its list of competence elements and fires the first one that is ready. In that case, the result of the competence element is returned. Otherwise, it returns FireResult(True, None) (this can never be returned by a competence element and is therefore uniquely identifyable).
