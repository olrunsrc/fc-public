Picks the given sense or sense-act from the given agent.

The method uses the agent's behaviour dictionary to get the sense / sense-act method.

The log domain is set to "[AgentId].Sense.[sense_name]".

The sense name is set to "[BehaviourName].[sense_name]".

Parameters:
  • agent (POSH.strict.Agent) - The agent that can use the sense.
  • sense_name (string) - The name of the sense
  • value (string) - The value to compare it to. This is given as a string, but will be converted to an integer or float or boolean, or left as a string, whatever is possible. If None is given then the sense has to evaluate to True.
  • predicate (string) - "==", "!=", "<", ">", "<=", ">=". If None is given, then "==" is assumed.
