 21      """Returns the type of the plan of the given plan file. 
 22       
 23      The type is returned as a string (e.g. 'SDC', 'RDC', SRDC'). If the type was not 
 24      found then an empty string is returned. The plan has to be given without 
 25      its file ending and has to be in the PLANPATH directory in the 
 26      corresponding library. 
 27       
 28      The function parses the plan line by line until it finds a plan type 
 29      identifier after a '('. This function can fail in several ways: 
 30          - There is a comment that has '(DC' or similar 
 31          - The bracket is on the line before the identifier 
 32          - Other ways that I haven't though about 
 33       
 34      @param planfile: Filename of the plan file 
 35      @type planfile: string 
 36      @return: Type of plan, or '' if not recognised 
 37      @rtype: string 
 38      @raise IOError: If plan file is not found. 
 39      """ 
 40      plan_lines = open(planfile, 'r').readlines() 
 41      for plan_line in plan_lines: 
 42          for plan_id in _plan_types: 
 43              # find plan type identifier 
 44              id_pos = plan_line.find(plan_id) 
 45              if id_pos == -1: 
 46                  continue 
 47              # is there a bracket before? 
 48              bracket_pos = plan_line[:id_pos].rfind('(') 
 49              if bracket_pos == -1: 
 50                  continue 
 51              # only valid if there is nothing else than whitespaces between 
 52              # the plan identifier and the bracket 
 53              if id_pos - bracket_pos == 1 or \ 
 54                 plan_line[bracket_pos + 1:id_pos].isspace(): 
 55                  return plan_id 
 56      return ''
