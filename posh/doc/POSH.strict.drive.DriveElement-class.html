isReady(self, timestamp)

source code 

Returns if the element is ready to be fired.

The element is ready to be fired if its trigger is satisfied and if the time since the last firing is larger than the one given by maxFreq. The time of the last firing is determined by the timestamp given to isReady when it was called the last time and returned True. This implies that the element has to be fired every time when this method returns True.

Parameters:
  • timestamp (long.) - The current timestamp in milliseconds
Returns: boolean
If the element can be fired.
