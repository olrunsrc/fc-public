Picks the given action from the given agent.

The method uses the agent's behaviour dictionary to get the action method.

The log domain is set to "[AgentId].Action.[action_name]".

The action name is set to "[BehaviourName].[action_name]".

Parameters:
  • agent (POSH.strict.Agent) - The agent that can perform the action.
  • action_name (string) - The name of the action
